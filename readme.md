# 📊 Sample Superstore Analysis

## 📌 Assignment Requirement

To proceed with this assignment, students must have successfully completed **Module 2 (Excel)**.
This is a **mandatory assignment** designed to strengthen practical knowledge of Excel.

---

## 🎯 Objective

The objective of this project is to analyze sales, profit, and regional performance using the Sample Superstore dataset and demonstrate practical implementation of Excel concepts.

---

## 🛠 Tools Used

* Microsoft Excel

  * Data Cleaning
  * Pivot Tables
  * Charts & Visualization
  * Dashboard Creation

---

## 📂 Files Included

* **Sample - Superstore.xls** → Raw dataset
* **Sample-Superstore-sol.xls** → Processed data & dashboard
* **dashboard_of_Superstore.png** → Dashboard preview

---

## 📈 Key Insights

* West region shows highest sales performance
* Technology category generates maximum profit
* Certain sub-categories are operating at a loss
* Profitability varies significantly across regions
